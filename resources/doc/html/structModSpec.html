std::string ModSpec::virtual_path

A constructed canonical path to represent this mod's location. This intended to be used as an identifier for a modpath that tolerates file movement, and cannot be used to read the mod files.

Note that mymod is the directory name, not the mod name specified in mod.conf.

Ex:

  • mods/mymod
  • mods/mymod (1) (^ this would have name=mymod in mod.conf)
  • mods/modpack1/mymod
  • games/mygame/mods/mymod
  • worldmods/mymod
