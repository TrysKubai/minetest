Detailed Description

A syncronization primitive that will wake up one waiting thread when signaled. Calling signal() multiple times before a waiting thread has had a chance to notice the signal will wake only one thread. Additionally, if no threads are waiting on the event when it is signaled, the next call to wait() will return (almost) immediately.
